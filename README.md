# FactCheck AI

자소서와 코드의 불일치(과장된 스펙)를 찾아내는 '허위사실 판독기'

기존 서비스는 'CS 암기 테스트'에 가깝지만, 우리 서비스는 자소서와 깃허브 코드를 교차 분석해서 경력 기술서의 진실성을 검증하는 **팩트 체크 면접관**

### 타겟

물경력/뻥튀기 스펙을 검증하고 싶은 취준생, 면접관

- **취준생:** "면접 가서 털릴까 봐 무섭다. 내 자소서가 과장되었는지 미리 알고 싶다."
- **면접관:** "GitHub 링크 냈길래 들어가 봤더니, 포크(Fork)만 해놓고 텅 비어 있거나 설정 파일만 있더라. 이거 거르는 데 시간 낭비하기 싫다."

### ChatGPT 쓰면 되지않나?

챗GPT로 하려면 프롬프트를 10줄 넘게 짜야 하고, 코드도 일일이 복사해서 붙여넣어야 함

→ 우리 서비스는 **버튼 하나로** 레포지토리 전체 구조를 파악해서 질문을 만듦

## 비슷한 타사 서비스

**1. 단순 AI 모의면접 (AI Interviewer)**:

자소서만 분석해서 "이 프로젝트에서 어려웠던 점은 무엇인가요?" 같은 일반적인 질문을 생성합니다. 코드를 안 보기 때문에 허위 사실을 걸러내지 못합니다.

예시: 각종 채용 플랫폼의 AI 역량 검사.

**2. 깃허브 분석기 (GitHub Analyzer)**:

커밋 수, 사용 언어 비율, 코드 스타일 등을 분석하여 "성실함"이나 "기술 스택"을 시각화합니다. 자소서 내용과의 일치 여부는 판단하지 않습니다.

예시: Gitfolio, 각종 Dev Ranking 사이트.

## 우리 서비스는 ..

"대용량 트래픽(Redis) 경험이 있다고 썼는데, 코드를 까보니 설정 파일만 있고 로직이 없다."

이 구체적인 불일치를 찾아내는 포인트가 기존 서비스들이 해결하지 못한 채용 담당자의 검증 비용을 낮추고, 취준생의 불안을 낮춥니다.
